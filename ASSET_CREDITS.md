# Asset Credits

MAZE TOWER does not use downloaded image, model, audio, or texture assets.

All current textures are procedural canvas textures generated at runtime in `src/render/textures.ts`:

- Grass and dirt ground texture
- Maze stone wall texture
- Tower brick texture
- Concrete stair texture
- Pedestal stone texture
- Door-frame wood texture

These procedural assets were created for this repository and do not require third-party attribution.
